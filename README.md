# dzpuke-notes

本仓库用于归档与发布多个独立的 HTML 页面。这些页面内容彼此无关，不针对任何特定网站或域名。

## 项目简介

`dzpuke-notes` 是一个以 HTML 静态页面为主的资料归档仓库。每个页面都可以独立访问，适合存放各类笔记、工具页面或演示文稿。仓库本身不提供搜索、评论或后台功能，所有页面均为纯前端实现。

## 目录说明

```
.
├── index.html          # 仓库入口页（导航列表）
├── pages/              # 存放所有 HTML 页面文件
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 共用资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的默认首页，通常包含页面列表或简要说明。
- `pages/`：所有独立 HTML 页面均放置在此目录下，文件名按编号或主题命名。
- `assets/`：存放页面共用的样式表、JavaScript 脚本或图片资源。

## 页面归档说明

- 每个 HTML 页面均为自包含的独立文件，可离线打开或在浏览器中直接运行。
- 页面之间不存在功能依赖或数据共享。
- 部分页面可能引用 `assets/` 目录下的公共资源，请确保路径正确。
- 页面内容可能随时间更新，历史版本不会单独保留，建议通过 Git 提交记录追溯。

## 维护说明

- 本仓库仅用于归档发布，不提供内容审核或修正渠道。
- 如需添加新页面，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的导航链接。
- 请勿在页面中嵌入外部追踪代码、广告或恶意脚本。
- 欢迎通过 Pull Request 提交页面修正或样式改进。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/dzpuke-notes.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 也可将仓库部署到任何静态托管服务（如 GitHub Pages、Netlify 等）以在线访问。
